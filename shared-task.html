Shared Task: Automatic Situation Report ("Sitrep") Generation

LT4CPR will host a shared task on automatic situation report ("sitrep") generation from crisis-related social media content. Situation reports are commonly compiled manually within the first 24 hours of a crisis and updated daily thereafter; partially automating this process can improve timeliness and coverage for situational awareness.

Task Overview

  • Input: crisis-related social media streams (e.g., posts/tweets) with metadata.
  • Output: a structured and/or narrative sitrep capturing key facts, impacts, and needs.
  • Evaluation: TBA (automatic + human evaluation; track(s) to be announced).
